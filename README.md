# QIU Lost & Found Management System
A full‑stack web application for digitizing the campus lost and found process at Quest International University. Students can report lost or found items, view all reports, track statuses, and manage their own submissions. Built with Node.js, Express, MySQL, and vanilla JavaScript.

## Features

- User registration and login (with password hashing)
- Submit lost or found item reports (with optional image upload)
- View all items with search and filter by type
- Detailed item view with contact information
- Personal dashboard to manage own reports (edit, delete, update status)
- Responsive design and custom 404 page

## Tech Stack
- Frontend: HTML5, CSS3, JavaScript (vanilla)
- Backend: Node.js, Express.js
- Database: MySQL
- Additional: Multer (file uploads), Bcrypt (password hashing), Express‑session

## Setup Instructions

### Prerequisites
- Node.js (v14 or later)
- MySQL server
- Git
